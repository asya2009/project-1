# LexiLing - Ein linguistisches Lexikon

Dieses Projekt ist ein einfaches linguistisches Lexikon mit Übersetzungen von deutschen Wörtern ins Russische. Es verwendet HTML, CSS und JavaScript, um Karten mit den deutschen Wörtern auf der Vorderseite und ihren russischen Übersetzungen auf der Rückseite anzuzeigen.

## Anleitung

1. Lade die Dateien `index.html`, `main.css` und `app.js` herunter.
2. Öffne `index.html` in einem Webbrowser deiner Wahl.
3. Du siehst eine Liste von Karten, auf denen deutsche Wörter auf der Vorderseite und ihre russischen Übersetzungen auf der Rückseite stehen.
4. Klicke auf eine Karte, um sie umzudrehen und die Übersetzung anzuzeigen.

## HTML-Dokument

Das HTML-Dokument (`index.html`) enthält die grundlegende Struktur der Anwendung. Hier sind einige wichtige Elemente:

- `<!DOCTYPE html>`: Die Deklaration des Dokumenttyps.
- `<head>`: Der Kopfbereich mit Metadaten und Verweisen auf CSS und JavaScript-Dateien.
- `<body>`: Der Hauptbereich, in dem die Karten angezeigt werden.

## CSS-Styling

Das Styling der Karten und des allgemeinen Layouts erfolgt über die CSS-Datei (`main.css`). Die Karten sind als Kartenpaare gestaltet, wobei die Vorderseite das deutsche Wort und die Rückseite die russische Übersetzung enthält.

## JavaScript-Interaktivität

Die Datei `app.js` enthält JavaScript-Code, der die Karten interaktiv macht. Wenn du auf eine Karte klickst, wird sie umgedreht, um die Übersetzung anzuzeigen. Dies geschieht mithilfe von Event-Listeners und DOM-Manipulation.
